O módulo Wi-Fi Wemos NodeMCU V3 ESP8266 CH340G com memória Flash extra de 32 MB é baseado no amplamente utilizado System-on-Chip (SoC) ESP8266 da Expressif. Adicionamos memória extra para 32 MB, o que é ideal para projetos maiores com bibliotecas extensas. Você não precisa se preocupar com o tamanho da memória.
Possui um conversor serial USB-TTL CH340G integrado e um conector Micro-USB.
A alimentação é de 4 a 9 V CC, conectada ao pino VIN.
Os pinos do conector são de 2,54 mm (0,1") e contêm 15 pinos x 2 linhas. Não é necessário soldar.

Compatível com redes Wi-Fi (IoT): Ponto de acesso, estação ou servidor web, etc.
Utiliza uma linguagem de programação simples baseada em LUA.
Compatível com a IDE do Arduino.
API orientada a eventos para aplicações de rede
9 pinos GPIO D0 – D8 com funcionalidade PWM, comunicação I2C e SPI.
Tensão de operação: 4,9 – 9 VCC
Dimensões: 49 x 26 x 5 mm (C x L x P) (sem os cabeçalhos)
Ele combina as funcionalidades de um ponto de acesso e estação Wi-Fi com um microcontrolador e utiliza uma linguagem de programação simples baseada em Lua.
O NodeM Wi-Fi ESP8266+32Mb oferece:
E/S de hardware semelhante ao Arduino 0. Pode ser programado na IDE do Arduino 0.
API orientada a eventos para aplicações de rede.
10 GPIOs (D0-D10), funcionalidade PWM, comunicação I2C e SPI, 1-Wire e ADC A0, etc., tudo em uma única placa.
Rede Wi-Fi (pode ser usado como ponto de acesso e/ou estação, hospedar um servidor web), conecta-se à internet para buscar ou enviar dados.
Para desenvolvimento, pode ser instalado em uma placa de ensaio.
